HOW TO USE
================================================================================

Local Development:
  1. cd /workspaces/yantrax-rl
  2. Backend: cd backend && python main.py
  3. Frontend: cd frontend && npm run dev
  4. Visit http://localhost:5173

Running Tests:
  Backend:  cd /workspaces/yantrax-rl && pytest -q
  Frontend: cd frontend && npm test --run

Deploying:

  • Run requests
