# YantraX RL Frontend

This directory contains the frontend application for YantraX RL, built with React and Vite. It provides a user interface to interact with the YantraX RL backend, displaying real-time market data, AI agent status, portfolio performance, and trading signals.

## Technologies Used

*   **Framework**: React
*   **Build Tool**: Vite
*   **Styling**: TailwindCSS
*   **Routing**: React Router DOM

## Getting Started

To run the frontend application:

1.  Navigate to the `frontend/` directory.
2.  Install dependencies: `npm install` or `yarn install`
3.  Start the development server: `npm run dev` or `yarn dev`
